Figma SVG → Lottie converter

Converts a CSS-animated SVG (Figma / Jitter export with @keyframes) into a Lottie JSON that lottie-react-native and lottie-web can play. Everything runs locally in your browser — nothing is uploaded.

Converts translate (bob), rotate, scaleX/Y (pulse) and opacity (fade) animations. Nested <g> groups are folded in — an animated group is baked into one layer that carries its motion. Paths inside <mask>/<clipPath>/<defs> are ignored. If a path command outside M L H V C Z is found, conversion stops — flatten arcs / quadratics first. Motion-path (offset-path) elements are placed statically.
